저격수[마피아]


경찰에게 수색당할 시 한번만 시민으로 나옵니다.
사격 : 지정한 대상을 제거합니다. 힘2 소모
정조준 : 대상을 무조건 제거합니다. 힘3 소모

연쇄살인마[마피아]


시작 후 힘이 8인 상태로 시작합니다. 매일밤 힘을 획득할 수 없습니다.
교란 : 다음 투표의 결과로 아무도 제거당하지 않습니다. 힘1 소모. 사용 후 필요한 힘이 2 증가합니다.
살인 : 대상을 제거하고 제거에 성공한 경우 전날낮 대상의 힘을 절반 가져옵니다. (절반 후 소숫점 반내림) 힘3 소모

독재자[마피아]


3일째 낮에 모두에게 독재자의 존재여부가 알려지며 그날은 투표하지 않습니다.
세뇌 : 낮이 되면 대상의 힘을 전부 소모시킵니다. 그후 대상은 힘을 획득할 수 없게 됩니다. 힘2 소모
몰살 : 낮이 되면 자신을 제외한 힘이 0인 대상들을 전부 제거합니다.

마담[간부]


*간부는 플레이어가 7인 이상일 때만 존재합니다.
경찰에게 수색당할 시 마피아라고 표시됩니다.*

마피아의 정체를 파악하고 시작합니다. (마피아는 자신의 정체 모름)
밀회 : 오늘밤 마피아는 힘 1을 얻습니다. 힘2 소모(일회용)
종언 : 다음낮을 생략하여 밤으로 넘어갑니다. 종언을 사용한 밤에 제거당할 시 효과가 없어집니다. 힘6 소모

밀렵꾼[간부]


*간부는 플레이어가 7인 이상일 때만 존재합니다.
경찰에게 수색당할 시 마피아라고 표시됩니다.*

마피아에게 제거당할 시 다른 플레이어와 함께 제거 당할 수 있습니다.
4일째 밤에 보름달이 뜨며 전체 플레이어가 이를 인식합니다.
포착 : 대상을 지정하여 대상은 다음날 밤에 힘을 획득할 수 없게 됩니다. 힘2 소모
사냥 : 보름달이 뜬 경우에만 사용 가능하며 오늘밤 마피아의 능력은 힘을 소모하지 않습니다.
또한 마피아의 공격은 대상을 무조건 제거합니다. 힘2 소모

배신자[간부]


*간부는 플레이어가 7인 이상일 때만 존재합니다.
경찰에게 수색당할 시 마피아라고 표시됩니다.*

마피아의 정체를 알며 직업 배정 시 해당 게임에 존재하지 않는 시민의 직업을 하나 알 수 있습니다.
은닉 : 낮에 사용 가능하며 다음날 밤 경찰의 수색에 시민이라고 나옵니다. (일회용)
배신 : 마피아 사망 시 힘6을 소모하여 자신이 마피아가 됩니다. 힘이 없다면 발동할 수 없습니다.
또한 이후 세번째 낮이 지나면 마피아가 승리합니다.

경찰


대상의 마피아여부를 알아냅니다. 힘3 소모

의사


힘이 3 이상일 시 단한번 제거당하지 않습니다. 자신은 이 사실을 알 수 있습니다.
치료 : 대상을 지정하여 다음낮에 대상은 제거당하지 않습니다. 힘2 소모(자신 치료 불가)

건달


공갈 : 대상을 지정하여 다음낮 대상은 힘이 1 감소합니다.
능력을 사용한 밤에 제거당할 시 발동되지 않습니다. 힘2 소모

정치인


투표로 사망 시 마피아는 힘을 1 얻습니다.
뇌물 : 다음낮 투표로 죽지 않습니다. 힘3 소모

탐정


마피아의 힘이 0이 될 시 마피아의 정체를 알아냅니다. 또한 마피아는 사전에 탐정의 존재를 알게 됩니다.
취조 : 오늘밤 대상의 능력사용여부를 알 수 있습니다.

군인


1일째 밤에 마피아의 능력 대상이 되지 않습니다. 이후 마피아는 다른 대상을 선택하게 됩니다.
정보 : 직업 하나를 지정하여 그 직업이 해당 게임에 존재하는지를 알아냅니다.

이상자


자신이 이상자인 것을 모른 채로 시작합니다. 행인을 제외한 시민의 직업 중 하나를 가짜로 받습니다.
가짜 직업은 능력도 가짜로 사용되며 다른 이들에게 능력이 인식되지 않습니다.
사망 시 자신의 정체가 모두에게 알려집니다.

행인


아무 능력이 없습니다.
